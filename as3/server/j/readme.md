`j` Join XT Packet Category
===========================

__XT Category__ : `j`

__XT Category Abbreviation__ : `Join Handlers`

Different files in this section represent different values of sub-category sent by client. 

## General Packet Structure 
```python
%xt%group-category% ROOM_INTERNAL_ID % REST_DATA/PARAMS %
```

__NOTE__ : For these packets, internal id is not recommended to be -1.
